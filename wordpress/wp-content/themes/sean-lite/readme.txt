Sean Lite WordPress Theme
---------------------

Sean Lite is a minimal and clean WordPress theme, for manage your personal blog, optimized for the mobile devices and based on Bootstrap framework.

Created by ThemeinProgress, http://www.themeinprogress.com 
Demo: http://www.wpinprogress.com/demo/sean
